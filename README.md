# Login_Form

A modern, interactive dual-panel Sign-In and Sign-Up interface featuring seamless CSS sliding animations. This project demonstrates responsive UI design and state-driven DOM manipulation, allowing users to toggle between logging in and registering without requiring a page reload.

## ✨ Features

* **Interactive Sliding Overlay:** Smooth, visually appealing transition effects when switching between Login and Registration views.
* **Modern UI/UX:** Clean, minimalist design utilizing a purple-and-white color palette with soft, rounded elements.
* **Social Authentication Layout:** Pre-built interface components for integrating Facebook, Twitter, LinkedIn, and Google logins.
* **Single-Page Experience:** Eliminates navigational friction by keeping the user on a single view for both authentication flows.

## 💻 Built With

* **Frontend:** React.js *(or HTML5 / Vanilla JavaScript)*
* **Styling:** Tailwind CSS *(or Custom CSS3)*
* **Icons:** FontAwesome
